**–°–≤—è–∑–∞–Ω–Ω—ã–µ —Ç–µ–º—ã:** [[–õ–∏–Ω–µ–π–Ω–∞—è —Ä–µ–≥—Ä–µ—Å—Å–∏—è]]

–†–∞–∑–±–µ—Ä—ë–º **–ø–æ—à–∞–≥–æ–≤–æ**, –∫–∞–∫ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å **—Ü–µ–Ω—Ç—Ä–∞–ª—å–Ω—É—é —Ä–∞–∑–Ω–æ—Å—Ç–Ω—É—é –ø—Ä–æ–∏–∑–≤–æ–¥–Ω—É—é** –¥–ª—è **–ø–æ–∏—Å–∫–∞ –ª–∏–Ω–µ–π–Ω–æ–π —Ñ—É–Ω–∫—Ü–∏–∏ $ y = ax + b $**, –∫–æ—Ç–æ—Ä–∞—è –∞–ø–ø—Ä–æ–∫—Å–∏–º–∏—Ä—É–µ—Ç –¥–∞–Ω–Ω—ã–µ –∏–∑ —Ç–∞–±–ª–∏—Ü—ã $(x_i, y_i)$.

## –¶–µ–ª—å –∑–∞–¥–∞—á–∏

–£ –Ω–∞—Å –µ—Å—Ç—å —Ç–∞–±–ª–∏—á–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ ‚Äî –ø–∞—Ä—ã $(x_i, y_i)$ —Å—Ç–æ–ª–±—Ü–æ–≤ (Y, X), –∏ –º—ã —Ö–æ—Ç–∏–º –Ω–∞–π—Ç–∏ —Ç–∞–∫—É—é **–ª–∏–Ω–µ–π–Ω—É—é —Ñ—É–Ω–∫—Ü–∏—é**:

$$
\hat{y}_i = a x_i + b,
$$

–∫–æ—Ç–æ—Ä–∞—è –Ω–∞–∏–ª—É—á—à–∏–º –æ–±—Ä–∞–∑–æ–º –æ–ø–∏—Å—ã–≤–∞–µ—Ç —ç—Ç–∏ –¥–∞–Ω–Ω—ã–µ. –î–ª—è —ç—Ç–æ–≥–æ –º–∏–Ω–∏–º–∏–∑–∏—Ä—É–µ–º **—Å—É–º–º—É –∫–≤–∞–¥—Ä–∞—Ç–æ–≤ –æ—à–∏–±–æ–∫**:

–ù–æ –º—ã **–Ω–µ –º–æ–∂–µ–º –≤–∑—è—Ç—å –∞–Ω–∞–ª–∏—Ç–∏—á–µ—Å–∫–∏–µ –ø—Ä–æ–∏–∑–≤–æ–¥–Ω—ã–µ**, –ø–æ—ç—Ç–æ–º—É –∏—Å–ø–æ–ª—å–∑—É–µ–º **—á–∏—Å–ª–µ–Ω–Ω–æ–µ –¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä–æ–≤–∞–Ω–∏–µ** (–≤ –¥–∞–Ω–Ω–æ–º —Å–ª—É—á–∞–µ ‚Äî **—Ü–µ–Ω—Ç—Ä–∞–ª—å–Ω—É—é —Ä–∞–∑–Ω–æ—Å—Ç–Ω—É—é –ø—Ä–æ–∏–∑–≤–æ–¥–Ω—É—é**) –¥–ª—è –ø–æ–∏—Å–∫–∞ –æ–ø—Ç–∏–º–∞–ª—å–Ω—ã—Ö `a` –∏ `b`.

## –û–±—â–∏–π –ø–æ—Ä—è–¥–æ–∫ –¥–µ–π—Å—Ç–≤–∏–π

### 1: –û–ø—Ä–µ–¥–µ–ª–∏–º —Ü–µ–ª–µ–≤—É—é —Ñ—É–Ω–∫—Ü–∏—é

–ú—ã —É–∂–µ —ç—Ç–æ —Å–¥–µ–ª–∞–ª–∏:

$$

–≠—Ç–æ –Ω–∞—à–∞ **—Ü–µ–ª–µ–≤–∞—è —Ñ—É–Ω–∫—Ü–∏—è –æ—à–∏–±–∫–∏**. –û–Ω–∞ –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –ø–∞—Ä–∞–º–µ—Ç—Ä—ã `a` –∏ `b`, –∞ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —á–∏—Å–ª–æ ‚Äî –º–µ—Ä—É –æ—à–∏–±–∫–∏ –ø—Ä–∏ —ç—Ç–∏—Ö –∑–Ω–∞—á–µ–Ω–∏—è—Ö.

### 2: –ù–∞–ø–∏—à–µ–º —á–∏—Å–ª–µ–Ω–Ω—ã–π –≥—Ä–∞–¥–∏–µ–Ω—Ç —á–µ—Ä–µ–∑ —Ü–µ–Ω—Ç—Ä–∞–ª—å–Ω—ã–µ —Ä–∞–∑–Ω–æ—Å—Ç–∏

–ì—Ä–∞–¥–∏–µ–Ω—Ç ‚Äî —ç—Ç–æ –≤–µ–∫—Ç–æ—Ä –∏–∑ —á–∞—Å—Ç–Ω—ã—Ö –ø—Ä–æ–∏–∑–≤–æ–¥–Ω—ã—Ö:

$$
\nabla S(a, b) = \left( \frac{\partial S}{\partial a}, \frac{\partial S}{\partial b} \right)
$$

–ß–∏—Å–ª–µ–Ω–Ω–æ –∏—Ö –º–æ–∂–Ω–æ –≤—ã—á–∏—Å–ª–∏—Ç—å —Ç–∞–∫:

$$

$$

$$

–ì–¥–µ `h` ‚Äî –º–∞–ª–µ–Ω—å–∫–æ–µ —á–∏—Å–ª–æ, –Ω–∞–ø—Ä–∏–º–µ—Ä `h = 0.00001`

### 3: –†–µ–∞–ª–∏–∑—É–µ–º –≥—Ä–∞–¥–∏–µ–Ω—Ç–Ω—ã–π —Å–ø—É—Å–∫

–ò—Å–ø–æ–ª—å–∑—É–µ–º –Ω–∞–π–¥–µ–Ω–Ω—ã–π –≥—Ä–∞–¥–∏–µ–Ω—Ç, —á—Ç–æ–±—ã –¥–≤–∏–≥–∞—Ç—å—Å—è "–≤–Ω–∏–∑ –ø–æ —Å–∫–ª–æ–Ω—É" –æ—à–∏–±–∫–∏:

- `learning_rate` ‚Äî —à–∞–≥ –æ–±—É—á–µ–Ω–∏—è (–Ω–∞–ø—Ä–∏–º–µ—Ä, `0.001`).
- –ò—Ç–µ—Ä–∞—Ç–∏–≤–Ω–æ –æ–±–Ω–æ–≤–ª—è–µ–º `a` –∏ `b`, –ø–æ–∫–∞ –Ω–µ —Å–æ–π–¥—ë—Ç—Å—è –∏–ª–∏ –Ω–µ –¥–æ—Å—Ç–∏–≥–Ω–µ–º –Ω—É–∂–Ω–æ–≥–æ —á–∏—Å–ª–∞ —à–∞–≥–æ–≤.

### 4: –ü—Ä–æ–≤–µ—Ä—è–µ–º —Ä–µ–∑—É–ª—å—Ç–∞—Ç

```python
# –¢–∞–±–ª–∏—Ü–∞ –¥–∞–Ω–Ω—ã—Ö (X ‚Äî –∞—Ä–≥—É–º–µ–Ω—Ç—ã, Y ‚Äî –∑–Ω–∞—á–µ–Ω–∏—è)
X = [3.47, 1.23, 7.82, 0.56, 5.01, 9.12, 2.34, 4.56, 6.78, 8.90, 0.12, 3.33, 7.00, 2.10, 5.55]
Y = [9.98, 4.38, 20.85, 2.70, 13.83, 24.10, 7.15, 12.70, 18.25, 23.55, 1.60, 9.63, 18.80, 6.55, 15.18]

# –®–∞–≥ –¥–ª—è —á–∏—Å–ª–µ–Ω–Ω–æ–≥–æ –¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä–æ–≤–∞–Ω–∏—è (–º–∞–ª–µ–Ω—å–∫–æ–µ —á–∏—Å–ª–æ)
h = 1e-5

# –°–∫–æ—Ä–æ—Å—Ç—å –æ–±—É—á–µ–Ω–∏—è (learning rate)

# –ù–∞—á–∞–ª—å–Ω—ã–µ –ø—Ä–∏–±–ª–∏–∂–µ–Ω–∏—è –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç–æ–≤ a –∏ b

# –ß–∏—Å–ª–æ –∏—Ç–µ—Ä–∞—Ü–∏–π –≥—Ä–∞–¥–∏–µ–Ω—Ç–Ω–æ–≥–æ —Å–ø—É—Å–∫–∞

def compute_loss(a, b):
    total_error = 0.0
    for x, y in zip(X, Y):
        prediction = a * x + b    # –ü—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏–µ –ø–æ —Ç–µ–∫—É—â–µ–π –ø—Ä—è–º–æ–π
        error = y - prediction    # –û—à–∏–±–∫–∞ (–æ—Å—Ç–∞—Ç–æ–∫)
        total_error += error ** 2 # –û—à–∏–±–∫–∞ (–æ—Å—Ç–∞—Ç–æ–∫)
    return total_error


# –í—ã—á–∏—Å–ª–µ–Ω–∏–µ –≥—Ä–∞–¥–∏–µ–Ω—Ç–∞ —á–∏—Å–ª–µ–Ω–Ω–æ —á–µ—Ä–µ–∑ —Ü–µ–Ω—Ç—Ä–∞–ª—å–Ω—ã–µ —Ä–∞–∑–Ω–æ—Å—Ç–∏
def numerical_gradient(a, b):
    # –ü—Ä–æ–∏–∑–≤–æ–¥–Ω–∞—è –ø–æ a: [S(a+h, b) - S(a-h, b)] / (2*h)

    loss_a_minus = compute_loss(a - h, b)
    grad_a = (loss_a_plus - loss_a_minus) / (2 * h)
    
    # –ü—Ä–æ–∏–∑–≤–æ–¥–Ω–∞—è –ø–æ b: [S(a, b+h) - S(a, b-h)] / (2*h)

    loss_b_minus = compute_loss(a, b - h)
    grad_b = (loss_b_plus - loss_b_minus) / (2 * h)
    return grad_a, grad_b


# –ì—Ä–∞–¥–∏–µ–Ω—Ç–Ω—ã–π —Å–ø—É—Å–∫

# –ü–æ—Å–ª–µ –∑–∞–≤–µ—Ä—à–µ–Ω–∏—è –≤—ã–≤–æ–¥–∏–º —Ñ–∏–Ω–∞–ª—å–Ω—ã–µ –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç—ã

print(f"y = {a:.4f} * x + {b:.4f}")
```
