Построим полную таблицу состояний для грамматики и входной строки `id + id`.

### Грамматика

```
S -> E + E | ε
E -> id
```

### Входная строка: `id + id`

### Шаг 1: Инициализация

Создаем таблицу Эрли `S` с 4 состояниями (длина строки + 1).

- Добавляем элемент `(S' -> • S, 0, 0)` в состояние `S[0]`.
